# APP Skill

Use this skill when working on the APP repository itself.

## Scope

- specification edits
- repository governance
- RFC authoring
- version snapshot management
- reference implementation planning

## Guidance

1. Treat `spec.md` as the working draft.
2. Treat `versions/` as released snapshots.
3. Use RFCs for substantive changes.
4. Keep the distinction clear:
   - AI-First Programming is the paradigm.
   - APP is the protocol.
